Getting Started

To get started with Dizzy, launch the Dizzy icon from the Start Menu (Windows) or your Desktop (Linux). Then select "Browse Help" from the "Help" menu. In the help browser window that appears, click on the "Getting Started" link in the table of contents pane on the left side. This will display the section of the application help scrren that explains in a step-by-step tutorial, how to begin using Dizzy.

Additional Information

A comprehensive User Manual is available for Dizzy. It is the file "UserManual.html" in the "docs" directory where you installed Dizzy.
